QU'EST CE QUE LE PHP

LE PHP EST UN LANGAGE DE PROGRAMMATION OPEN-SOURCE INTERPRETE COTE SERVEUR,ESSENTIELLEMENT UTILISE POUR LA CREATION DE PAGE WEB DYNAMIQUE. L'ABREVIATION INITIALE SIGNIFIAIT (PERSONAL HOME PAGE TOOLS ) MAIS EST MAINTENANT UTILISE COMME ACRONYME POUR < PHP: HYPERTEXT PREPROCESSOR>. SI LES LANGAGES COTES CLIENTS TELS QUE HTML, CSS OU JAVASCRIPT NE SONT INTERPRETES PAR LE NAVIGATEUR WEB QUE LORSQU'UNE PAGE WEB EST OUVERTE, LE CODE PHP EST DEJA EXECUTE SUR LE SERVEUR WEB. LES SCRIPTS PHP GENERENT ALORS UNE SORTIE HTML,QUI EST ENVOYE AU NAVIGATEUR WEB. CELUI-CI N'OBTIENT PAS LE CODE REEL MAIS SEULEMENT LE RESULTAT DE L'EXECUTION DU SCRIPT. LE PRINCIPAL CHAMP D'APPLICATION DE PHP EST LA PROGRAMMATION COTE SERVEUR, EN PARTICULIER LES SITES WEB ET LES APPLICATIONS DYNAMIQUES. IL EST EGALEMENT ADAPTE A LA CREATION D'APPLICATIONS BUREAUTIQUES ET A LA PROGRAMMATION EN LIGNE DE COMMANDE. BIEN QU'IL SOIT ADAPTE AUX DEBUTANTS, PHP OFFRE UNE GAMME IMPRESSIONNANTE DE FONCTIONNALITES. LE LANGAGE DE PROGRAMMATION SE CARACTERISE PAR UN VASTE SUPPORT DE BASES DE DONNEES; IL PEUT ETRE UTILISE SUR PLUSIEURS PLATEFORMES, ET IL EST DISPONIBLE SOUS LICENCE PHP specifique, CE QUI PERMET D'UTILISER ET DE MODIFIER LE CODE SOURCE.
